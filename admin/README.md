# Cultural Hatti Admin Panel

Admin dashboard for managing Cultural Hatti e-commerce platform.

## Features

- Admin authentication
- Dashboard with analytics
- Product management
- Order management
- User management

## Setup

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```

Default admin credentials:
- Email: admin@culturalhatti.in
- Password: Admin@1234
